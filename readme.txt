Write a program to display a transparent magnification ball, that bounces around the desktop screen.  The program should be implemented as a full-screen transparent overlay, over the running Windows desktop, and the ball magnifies whatever is on the screen underneath it.

Allow the program to continue to run whilst the user interacts with the regular desktop behind.  Clicking the mouse on the ball should terminate the program.

Add logging to standard out, and when running the program use a timeout of 15 seconds.

Use the library egui (and any dependencies), and no other libraries except for the Rust standard library.
